=======================================================================
Willow builds out each model and uses attributes to indicate each field
property.
For example:
#[ApplyModelColumnAttribute('Id', 'int', null, ['PK', 'NN', 'AI'])]
#[ApplyModelColumnAttribute('UserId', 'int', null, ['NN'])]
#[ApplyModelColumnAttribute('Notes', 'string', 500)]

The first indicates that Id is an int a primary key, can't be null and
is auto incremented.
UserId is also an int and cannot be null.
Notes is a string with a length of 500 characters.

These attributes are used to validate insert and update (POST) requests.
